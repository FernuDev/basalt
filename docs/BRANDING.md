# Basalt — Guía de marca

Esta guía define la identidad visual y verbal de **Basalt**, el cliente de escritorio para PostgreSQL orientado a desarrolladores.

---

## Nombre y posicionamiento

- **Nombre:** Basalt  
- **Tagline:** *A fast, minimal PostgreSQL desktop client built for developers.*  
- **Subtítulo:** *Native performance. Clean interface. Zero bloat.*

Basalt evoca solidez y minimalismo (como la roca ígnea), velocidad y enfoque en lo esencial.

---

## Colores

| Uso        | Valor     | Ejemplo |
|-----------|-----------|--------|
| **Principal** | `#7C4FD4` | Botones, acentos, badges, links |
| **Texto**     | Negro / Gris según contraste | Contenido y UI |
| **Fondo**     | Blanco / tonos neutros | Fondos de la app |

El color principal (`#7C4FD4`) debe usarse para elementos interactivos, estados activos y refuerzo de marca en documentación y marketing.

---

## Tipografía

- **Títulos / UI:** [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk)  
- **Cuerpo / Interfaz:** [Jura](https://fonts.google.com/specimen/Jura)  
- **Código / SQL:** [Geist Mono](https://vercel.com/font)

Mantener esta combinación en la app y en materiales oficiales (landing, docs, capturas) para una identidad coherente.

---

## Logo

- **Ubicación del recurso:** `branding/logo.png` (referenciado desde el repositorio).
- **Uso:** Preferible sobre fondos claros; en fondos oscuros usar una variante clara si existe.
- **Espacio mínimo:** Dejar un margen de respiro alrededor del logo (mín. ~20% del alto del logo).

---

## Tono de voz

- **Directo y técnico:** Lenguaje claro, sin rodeos.
- **Orientado a desarrolladores:** Terminología precisa (conexión, schema, query, relaciones).
- **Minimalista:** Frases cortas; evitar marketing excesivo.

Ejemplos: *“No Electron. No overhead. Just a fast, focused tool.”* / *“Zero bloat.”*

---

## Uso en repositorio y redes

- **GitHub:** Badges y enlaces en README con el color `#7C4FD4`.
- **Releases / anuncios:** Mantener el mismo tagline y paleta para reconocibilidad.
